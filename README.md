Loafer is a boilerplate Sinatra app set up with:

- Haml
- Markdown
- Sass/Compass
- CoffeeScript
- Foreman

To use it:

```
git clone git://github.com/zeke/loafer.git new_hotness
cd new_hotness
rm -rf .git
bundle
foreman start
```

You're up and running at [localhost:5000](http://localhost:5000).
